## 示例原则

- 示例优先表达对象边界，不追求一次写满所有字段。
- YAML 可以被 validator 使用，但 v0.1 阶段仍可能随 schema 调整。
- 示例中出现的 hash、时间和路径是占位值。
- 执行前应先 `validate` 和 `explain`，再考虑 `dry-run` 或 runtime 执行。
